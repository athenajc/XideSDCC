SDCC MCU IDE

Integrated Development Environment of Small Device C Compiler

Version : 0.0.1
  • Integrate Editor, Compiler and Simulators
  • Target for all device supported by SDCC - 8051 family, PIC14/PIC16, Z80, DS390, DS400
  • Compiler supported - 8051 family, PIC14, PIC16, Z80, DS390, DS400
  • Simulator supported - 8051, PIC14, PIC16(on going)

Windows System requirements:

  • SDCC - http://sdcc.sourceforge.net/index.php#Download
  • GPUTILS - If you work with PIC14/PIC16 microcontroller http://sourceforge.net/projects/gputils/

Linux System requirements:

  • Python 2.7.5 - http://www.python.org/getit/releases/2.7.5/
  • wxPython 2.9.5 - http://www.wxpython.org/download.php#devel
  • SDCC - http://sdcc.sourceforge.net/index.php#Download
  • GPUTILS - If you work with PIC14/PIC16 microcontroller http://sourceforge.net/projects/gputils/
